# Women Endocrine Intelligence System

## Overview

The Women Endocrine Intelligence System is a comprehensive digital health platform designed specifically for women to track, understand, and manage their endocrine health. Built with modern web technologies, this application empowers users with personalized insights into their menstrual cycles, hormonal patterns, and overall well-being.

## Features

### 🔐 Secure Authentication
- User registration and login
- Secure password hashing
- Personalized user profiles

### 📊 Cycle Tracking
- Daily health entries including cycle phase, flow intensity, pain levels
- Mood, energy, sleep, and stress tracking
- Symptom logging with multi-select options
- Historical data visualization

### 🎓 Learning Modules
- Educational content about women's health and endocrinology
- Progress tracking for completed modules
- Interactive learning paths

### 👩‍⚕️ Doctor Directory
- Find healthcare professionals by specialty and location
- Contact information for medical consultations

### 💬 AI-Powered Chat Support
- Intelligent chatbot for health-related queries
- Personalized advice based on user data

### 🏆 Rewards System
- Gamification elements to encourage consistent tracking
- Achievement badges and milestones

### 🌐 Multilingual Support
- Support for multiple languages (English, Nepali)
- Localized user interface

## Technology Stack

- **Frontend**: React with TypeScript, Vite
- **UI Components**: Radix UI, Tailwind CSS
- **Backend**: Node.js with Express
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: JWT tokens
- **State Management**: TanStack Query

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- PostgreSQL database
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Aishwaryagithubit/Women-Endocrine-Intelligence-System.git
   cd Women-Endocrine-Intelligence-System
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up the database:
   ```bash
   npm run db:push
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser to `http://localhost:3000`

## Usage

1. **Onboarding**: Complete the initial setup with your cycle information
2. **Daily Tracking**: Log your daily health metrics
3. **Learning**: Explore educational modules about women's health
4. **Consultation**: Find doctors and get professional advice
5. **Chat**: Ask questions to the AI assistant
